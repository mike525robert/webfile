How to Setup



Step1: Upload horizon.zip to your host and unzip.
Step2: Create mySQL Database and Database User.
Step3: Open your browser and visit http://your-sitename/horizon
Step3: to register new customer visit http://your-sitename/horizon/enroll.php
Step4:open library/config.php and enter database details
Step4:open admin/admin-config.php and enter database details
Step5:open library/mail.php and set smtp email e.g. demo@finalyearproject.website
Step5:open view/process.php and set the internal support receiving email e.g. demo@finalyearproject.website
Step6: Login to admin panel http://your-sitename/horizon/admin with Username: admin & Password: admin1010 to change Website.

How To Edit Code?


Once You Install The Product on Your Server, You Will have 10 Folders and php Files.

On Folder "assets", We keep all kind of asset like CSS, JS, Images. So if You Want To Edit Or Change Any Asset, Have a Look on This Folder. the assets folder contains the user dashboard files and admin files (js, css, images)
on the images folder, we keep all the files related to login.php, pin.php and thumbnails where users profile pictures are stored.
on the "reg-assets" folder we keep or the files related to registration page (js, css, images, fonts, bootstrap e.t.c)
on the "database-files" contains the database file
on the "view" folder contains the user php functions and pages (otp, fund transfer e.t.c)
